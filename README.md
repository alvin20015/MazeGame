# MazeGame迷宫游戏代码文档
一、功能概述
本 Java 程序实现了一个简单的迷宫游戏，玩家可以通过输入 W、A、S、D 来控制角色在迷宫中移动，使用 M 查看当前地图，使用 Q 退出游戏。迷宫数据从文件中加载，程序会对迷宫文件进行格式验证，确保迷宫有正确的尺寸、唯一的起点和终点。
二、代码结构
代码主要包含以下几个部分：
常量定义：定义了迷宫中不同元素的字符表示，以及迷宫的最小和最大尺寸。
迷宫数据结构：使用二维字符数组存储迷宫，同时记录迷宫的宽度、高度、玩家的初始位置和游戏结束标志。
迷宫加载：从文件中读取迷宫数据，并进行格式验证。
用户输入处理：处理用户的输入，根据输入控制玩家移动或执行其他操作。
玩家移动：验证玩家的移动是否合法，并更新玩家的位置。
地图显示：显示当前迷宫地图，标记出玩家的位置。
游戏主循环：控制游戏的流程，直到玩家退出游戏或到达终点。
